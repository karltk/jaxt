The package contains JAxT support interfaces, which serve as markers for the JAxT tool.

Any class implementing a subtype of Axioms are considered to contain axioms. This is necessary for the JAxT support system in identifying axiom classes, but also gives the user an easy way to recognise them as well: looking at the javadoc generated list of all classes implementing the interface jaxt.framework.Axioms is an easy way to check if all axiom classes have been declared.

The marker AxiomSet<XA> indicates the the optional axioms in XA should be satisfied.

See the jaxt overview file for information about declaring and using axiom classes.
