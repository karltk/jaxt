The package jaxt contains JAxT framework support, axioms for Java standard classes and some example packages.
  • Subpackage jaxt.framework contains interfaces which JAxT uses to identify axiom classes.
  • Subpackages of jaxt.example contain example user code exploiting JAxT.
  • Subpackages of jaxt.java contain JAxT axioms for the Java standard packages.
See the jaxt overview file for information about declaring and using axiom classes.
